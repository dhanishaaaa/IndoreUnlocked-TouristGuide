# Indore Unlocked 🌆

## Project Title

**Indore Unlocked – A City Exploration & Tourism Guide Website**

---

# Brief Description

Indore Unlocked is a web-based city guide and tourism project designed to help users explore famous places in Indore. The website provides information about cafes, restaurants, temples, shopping places, street food, tourist attractions, malls, and entertainment spots in Indore.

The project allows users to:

* Explore different categories
* Search places dynamically
* Use chatbot suggestions
* View place details
* Open locations directly in Google Maps

The website is designed with an interactive and visually appealing user interface for better user experience.

# Installation / Execution Steps

## Step 1

Download or clone the project folder.

---

## Step 2

Open the project folder in VS Code.

---

## Step 3

Import the SQL database file into MySQL.

---

## Step 4

Install required Python packages.

```bash
pip install flask
```

---

## Step 5

Run the Flask application.

```bash
python new.py
```

---

## Step 6

Open the localhost link in browser.

Example:

```bash
http://127.0.0.1:5000
```
